# Тестовая документация для проекта dbWebTestAutomation

Проект **dbWebTestAutomation** использует Python, Playwright и Pytest для автоматизированного тестирования веб-сайта[https://www.demoblaze.com/]. Эта документация описывает тесты, которые были написаны для этого проекта.

## Проверка заголовка домашней страницы

*Описание:* Тест проверяет, что заголовок домашней страницы соответствует ожидаемому значению в зависимости от типа теста (позитивный или негативный).

*Предусловия:* 
- Имеется доступ к интернету.

*Шаги выполнения:*
1. Открыть домашнюю страницу по заданному URL.
2. Проверить, что заголовок страницы соответствует ожидаемому значению в зависимости от типа теста.

*Ожидаемые результаты:*
- В случае позитивного теста (`test_type='positive'`), заголовок страницы должен быть равен ожидаемому заголовку.
- В случае негативного теста (`test_type='negative'`), заголовок страницы не должен быть равен ожидаемому заголовку.

*Постусловия:* 
- Состояние системы восстановлено после выполнения теста.

**Тест-кейс:**

*ID тест-кейса:* TC001

*Описание:* Проверка заголовка домашней страницы при позитивном тесте.

*Предусловия:* 
- Имеется доступ к интернету.

*Шаги выполнения:*
1. Запустить тест с параметрами: 
   - URL: 'https://www.demoblaze.com/'
   - Ожидаемый заголовок: 'STORE'
   - Тип теста: 'positive'.
2. Проверить, что заголовок страницы равен ожидаемому значению 'STORE'.

*Ожидаемый результат:* Заголовок страницы соответствует ожидаемому значению.

*Постусловия:* 
- Состояние системы восстановлено после выполнения теста.
